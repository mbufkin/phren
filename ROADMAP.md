# Phren — Roadmap

## Now

Single-user mode is built and proven (chess, London System, document upload). Maintain and harden.

- [x] Upload documents → AI-generated interactive course
- [x] Lesson playback: teach → move → check
- [x] Deterministic mastery scoring
- [x] AI coaching review (strengths, focus, next steps)
- [x] Drill mode for missed concepts
- [x] Progress tracking with learner profile

## Next

- Domain expansion: validate with more document types (legal, medical, technical)
- Better PDF extraction — tables, diagrams, structured content
- Spaced repetition — schedule review lessons based on mastered/weak concepts
- Export formats: printable summaries (PDF), shareable progress reports

## Later

- Multiple document uploads in a single course (cross-reference)
- Custom domain helpers for specific subjects
- Voice input for free-response answers
- Browser extension — one-click "learn this page"

## Not Yet

- Classroom/teacher mode — that's Choros
- Multi-student management — that's Choros
- Curriculum mapping — that's Choros
- Authentication or accounts — local-first, no sign-up
- Mobile native app — web-only, responsive
